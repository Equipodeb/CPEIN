Introduccion a HTML

Como se menciono en la anterior pagina sobre el curso de programacion en html 1, html funciona con etiquetas de apertura y cierre, mismas etiquetas que si no concluimos de forma correcta no nos marcara un error, pero sin embargo te mostrare como es que si utilizamos por ejemplo una etiqueta p en una h1 puede mal formatear el texto

Hola soy una etiqueta p sin cierre

y yo una h1 con cierre

Como puedes darte cuenta las etiquetas tienen una clonacion, por ejemplo la etiqueta html tiene la clonacion de su misma etiqueta pero con un / entre la <

< /html>

Codigo erroneo

< p>

Notese la etiqueta p de apertura y notese que la etiqueta de cierre no esta

>

Codigo correcto

< p>

Notese la etiqueta de apertura y cierre

< /p>

Cuando nosotros ponemos una etiqueta equivocada como en el caso de la etiqueta p va a surgir un error, pero eso lo veremos en las siguientes lecciones.
